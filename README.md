# 강동윤 개인 프로필 웹사이트

반반운동센터 운동지도사 강동윤의 개인 프로필 홈페이지입니다.

## 기술 스택

- **React 19** + **TypeScript**
- **Vite** — 빌드 도구
- **Tailwind CSS v4** — 스타일링
- **Framer Motion** — 애니메이션
- **Vercel** — 배포

## 주요 섹션

- 히어로 (이름, 직함, 프로필 사진)
- 학력 (Educational)
- 경력 (Professional)
- 자격증 (Certificate)
- 연락처 (Contact)

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
